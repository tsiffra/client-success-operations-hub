# Client Success Operations Hub

A full-stack support operations dashboard built with React, Express, and SQLite.

It imports customer support tickets from a CSV dataset and provides:

- Dashboard with SLA metrics
- Ticket management
- Customer accounts
- Analytics
- Search and filtering

## Tech Stack

- React
- Express
- SQLite
- Vite

## Running locally

npm install
npm run import
npm run dev

## Dataset

Based on a public helpdesk ticket dataset from Kaggle.
